Formularios

Ideas Principales

  • Los formularios HTML son elementos de las páginas web que nos sirven para recuperar información del usuario y enviarla al servidor para que pueda ser procesada
  • Los campos de entrada son los elementos básicos de un formulario ya que son los que nos permiten recuperar información del usuario de diferentes formas.

Apuntes Tomados

  • Campos de texto
  • Constraseñas
  • Checkbox
  • Radios
  • Botonos de envio
  • Botones de borrado
  • Ficheros
  • Campos Ocultos
  • Imagenes
  • Botones
  • Calendarios

Resumen

Permiten que los usuarios ingresen datos, como nombres, direcciones de correo electrónico o mensajes, y los envíen a un servidor para su procesamiento. Estos datos pueden ser utilizados para diversas finalidades, como realizar compras en línea, suscribirse a boletines, enviar mensajes de contacto, entre otros.

Tablas

Ideas Principales

  • Una tabla en html consta de celdas de tabla dentro de filas y columnas
  • Las tablas HTML nos permiten organizar los datos en filas y columnas

Apuntes Tomados

  • La tabla permite buscar con rapidez y facilita los valores entre tipos de datos
  • Sus elementos: thead, th, tbody, tr, td, caption, tfoot
  • El rawspan indica el numero de filas que ocupara la celda
  • El colspan indica el numero columnas que ocupara la celda

Resumen

La etiqueta table en HTML se utiliza para estructurar datos en forma de filas y columnas, creando lo que conocemos como una tabla. Es una forma eficaz de organizar información de manera legible y accesible.

Especificidad

Ideas Principales

  • La especificidad es el medio por el cual los navegadores deciden que valores de propiedad css son los más relevantes para un elemento y por lo tanto se aplicaran
  • Cuando varias declaraciones tienen la misma especificidad, la ultima declaracion encontrada en css se aplica al elemento
  • Si la ultima regla cuenta con la misma regla se escribe dos veces de estilo externa entonces la regla inferior en la hoja de estilo esta más cerca del elemento que se va a diseñar
  • El selectro universal y los valores heredaos tienen una especificidad de 0
  • Al usar la regla important anula cualquier regla de estilo y esta interactua directamnete con el estilo

Apuntes Tomados

  • Cada selectro tiene su lugar en la jerarquia y se definen en niveles
  • Estilos inline
  • Id's
  • Clases, atributos y pseudo-clases
  • Elementos y pseudo-elementos

Resumen

Basicamente es una regla que los navegadores utilizan para determinar qué estilo se aplica a un elemento cuando varias reglas compiten por el mismo. Es decir, cuando tienes múltiples declaraciones CSS que podrían aplicarse a un mismo elemento, el navegador necesita una forma de decidir cuál de ellas "gana".

After y Before

Ideas Principales

  • Son pseudo elementos que perimiten insertar contenido en una pagina desde css sin qie tenga que estar en el HTML
  • El reusltado final no esta realmente en el docuemnto, aparecera en la pagina como si lo estuviera
  • Los valores del content pueden ser una cadena de caracteres, una imagen y insertarlas en sus dimensiones exactas , y unos para insertar formas y lograr crear figuras

Apuntes Tomados

  • Se debe de udar la propiead de content
  • El selector after inserta algo despues del contenido de cada uno de los elementos seleccionados
  • El selector before inserta algo antes del contenido de cada uno de los elementos seleccionados

Resumen

En resumen son herramientas que te permiten agregar contenido antes o después del contenido principal de un elemento HTML, sin necesidad de crear un elemento extra y dan mucha flexibilidad para personalizar el diseño de tus páginas web y crear efectos visuales interesantes.

Colores en CSS3

Ideas Principales

  • Formas de representar colores: Keywords, Hexadecimal, RGB y HSL
  • Seleccionar la paleta de colores correcta es importante para comunicar un mensaje
  • Realizar pautas generara un diseño armónico hacia la pagina
  • Se deben de utilizar colores de la misma gama cromática, complementarios y opuestos
  • Se debe de tener cuidado con ciertos colores, ya que ciertos colores estan relacionados con normas o fundamentos culturales y uno como usuario los asocia rápidamente

Apuntes Tomados

  • SRGB (Standard Red, Green, Blue)
  • Elegir la paleta errónea provocara que el usuario huya de nuestra pagina
  • No sobre exagerar con el uso de los colores en una pagina web
  • HUE
  • Saturacion
  • Brillo

Resumen

La elección de colores para crear una página es una decisión delicada ya que es la generara el tema al que se enfoca, la experiencia del usuario y el cómo hará para que usuario no sienta una sobre saturación de colores y el también que esta misma no se sin vida

Header

Ideas Principales

  • Logotipo: Sirve como guía visual para el usuario y permite reforzar la imagen de la marca
  • Nav: Permite al usuario la navegación entre distintas secciones
  • Buscador: Se debe de tener un vasto conocimiento de lo que tienes en tu página web para colocarlo, pero es para buscar
  • Datos de contacto: No se deben de colocar demasiados y esto se puede colocar en el footer
  • Barras de ligas: Sirven para mostrar anuncios de descuentos o llamadas de atención para el usuario
  • Redes sociales: Son un canal complementario y sirven para el trafico de la web

Apuntes Tomados

  • El header es un a de las partes mas importantes de una página web, ya que es el elemento que los usuarios visualizan en primer lugar cuando llegan a un proyecto
  • Debe de estar ordenada y limpia
  • Tendrá en el mismo diseño para todas las paginas de un sitio web

Resumen

Los elementos que deberían de llevar o pueden llevar son: Logotipo, menú, buscador, datos de contacto, redes sociales, carrito de compras, botones destacados, selector de idioma etc

Footer

Ideas Principales

  • Es el pie de página de una sección y normalmente es olvidad en muchos proyectos web pero bien diseñado puede ser muy útil y efectivo
  • Redes sociales: Es uno de los elementos que es recomendable colocar en esta sección
  • Copyright: Son los créditos del diseñador y los autores del diseño grafico
  • Datos de contacto: Son útiles para que las vistas se pongan en contacto con la empresa o negocio
  • Los textos legales obligatorios son: la política de privacidad y el aviso legal
  • Las cookies: deben de aparecer en forma de aviso de barra o mensaje

Apuntes Tomados

  • Es un espacio genérico que suele tener el mismo diseño en todas las páginas del sitio web
  • Debe contener los avisos legales obligatorios de toda la web
  • Cosas extras que son importantes dependiendo de la pagina web: Sellos de confianza, CTA (Call To Action), Sitemap y Newsletter

Resumen

En resumen el footer es una parte esencial de cualquier página web que proporciona información valiosa a los usuarios y mejora la experiencia de navegación.
